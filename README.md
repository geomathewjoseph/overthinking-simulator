## ğŸ’¡ Origin Story

*"It's funny because it sounds exactly like you spiraling."* - My two best friends, calling me out.

This project was born out of a particularly chaotic few weeks in my personal life. I found myself spiraling over every little decision (should I send this text? does that emoji look passive-aggressive?), and somewhere in the middle of a 3 AM worrying session, I thought: *What if AI could do this for me? Could I automate my worst-case scenarios?*

It started as a joke. I built a quick prototype and showed it to my two best friends, expecting a chuckle. Instead, they laughed and told me it sounded exactly like me spiraling. (And yeah, lol, it is true). I shared it with a few more friends, and the reaction was unanimous - it wasn't just funny, it was painfully relatable.

That's when I realized the potential here. Our brains are often just being drama queens. By outsourcing that drama to an app, we can see these spirals for what they really are: hilarious, absurd, and usually completely unfounded.

This simulator is a lighthearted way to hold a mirror up to our own overthinking. If an app can do it better than we can, maybe we can laugh at the absurdity of it all and finally let go.

### Manual Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/geomathewjoseph/overthinking-simulator.git
   cd overthinking-simulator
   ```

2. **Get an API Key**
   - Sign up at [OpenRouter](https://openrouter.ai)
   - Create an API key at [openrouter.ai/keys](https://openrouter.ai/keys)

3. **Deploy to Vercel**
   ```bash
   npm i -g vercel
   vercel
   ```

4. **Add Environment Variable**
   - Go to your Vercel project settings
   - Add `OPENROUTER_API_KEY` with your API key
